# 📰 NewsMag - React News App

A modern and responsive news application built with React that fetches the latest headlines from NewsAPI and displays them in an elegant card-based interface.

## 🚀 Features

- 📰 Latest news headlines
- 📂 Category-based news filtering
- 🌐 Real-time news fetching using NewsAPI
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- 🔗 Direct links to full articles
- ⚡ Fast loading with React Hooks

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Bootstrap
- NewsAPI
- HTML5
- CSS3
