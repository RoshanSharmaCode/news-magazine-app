# 📰 NewsMag - React News App

A modern and responsive news application built with React that fetches the latest headlines from NewsAPI and displays them in an elegant card-based interface.

## 🚀 Features

- 📰 Latest news headlines
- 📂 Category-based news filtering
- 🌐 Real-time news fetching using NewsAPI
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- 🔗 Direct links to full articles
- ⚡ Fast loading with React Hooks

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Bootstrap
- NewsAPI
- HTML5
- CSS3

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/react-news-portal.git

```
2. Navigate to the project folder

```bash
cd react-news-portal
```

3. Install dependencies

```bash
npm install
```

4. Create a ```.env``` file

```bash
VITE_API_KEY=your_news_api_key
```

5. Start the development server

```bash
npm run dev
```
